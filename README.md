# 羊羊工艺 Miecraft

## 简介

羊羊，世界上一种普普通通的生物。经羊羊科学家最新的研究发现，他们身体里拥有与众不同的强大能量。而它们身上多彩的颜色，正是它们体内力量的体现。羊羊科学家们不断地研究，终于发现了和平利用羊羊能量的方法。

### 羊羊介绍

* 白
    * 纯洁的白羊羊拥有最纯净的羊羊能量，他们拥有无限的可能性。
* 橙
    * 天生活力的橙羊羊让人一看到就满心欢喜，充满干劲。
* 紫红
    * 紫红羊羊看似孤僻，心中可有万千烟花在绽放。
* 海蓝
    * 以海蓝羊羊的天赋，任何精彩画面它都不会错过。
* 黄
    * 黄羊羊好似羊群中的大哥，作为大家的榜样时刻关照着大家。
* 柠
    * 柠檬羊羊凭借刻苦的学习，在植物学上造诣颇深。
* 粉
    * 粉羊羊羊如其名，光是看到它就感到被治愈了。
* 灰
    * 灰羊羊我脾气爆，我不微笑你别闹。
* 浅灰
    * 浅灰羊羊以白羊羊为榜样，却怎么也追不上它。
* 青
    * 青羊羊磕cp磕的上瘾，没想到自己竟有此等魔力。(???)
* 紫
    * 作为羊羊们中学历最高的一员，它的大发现可谓惊天动地。
* 蓝
    * 和水相性最好的蓝羊羊对水的掌控是游刃有余。
* 棕
    * 凭借棕羊羊的外表，它获得了年度最佳反派的称号。
* 绿
    * 在植物学上拥有极高天赋的绿羊羊对树木结构最为清楚。
* 红
    * 红羊羊在探索地狱时获得了撒旦的力量，能够操控各种金属元素。
* 黑
    * 黑羊羊每次都神不知鬼不觉的出现，又神不知鬼不觉的消失。

## 物品设定

### 羊羊大百科魔法全书
#### 介绍
羊羊科学家们研究成果的结晶，只要拥有了它就能掌控羊羊们体内的能量。

#### 获得方式
合成一本书，丢给羊羊，它们吃掉后会吐出包含自身颜色的书。16种颜色集齐后便成为羊羊大百科全书。

#### 用途
用于羊羊魔法物品的能量注入。




### 羊羊魔法罐
#### 介绍
能够保存并转化羊羊能量的魔法罐。仅能够吸收并存储特定颜色的羊羊。

#### 获得方式
搭建3\*4\*3多方块结构，由最顶层至底层结构如下：


```
阶 阶 阶
阶 讲 阶
阶 阶 阶
```

```
玻 玻 玻
玻 毛 玻
玻 玻 玻
```

```
玻 玻 玻
玻 特 玻
玻 玻 玻
```

```
玻 玻 玻
玻 玻 玻
玻 玻 玻
```

其中：讲->讲台，阶->任意台阶（台阶高的一面向内，外低内高），玻->玻璃，毛->对应颜色羊毛，特->对应特殊方块（见下）。

把羊羊大百科魔法书放到顶部讲台即可合成，不同颜色合成需求及功能见下。

#### 用途
不同颜色的羊羊魔法罐用途均有不同，在使用前需要用羊羊魔法罐右击对应颜色羊羊来吸收他们。

* 白色
    * 特殊方块：骨块
    * 合成所需羊羊能量：无
    * 用途：
        * 最基础且纯净的羊羊魔法罐
        * 吸收白羊羊并转化为10点白色羊羊能量保存在罐内
        * 最多保存100点白色羊羊能量
* 橙色
    * 特殊方块：干草块
    * 合成消耗羊羊能量：100点白色羊羊能量 50点柠檬羊羊能量
    * 用途：
        * 橙色是成长的颜色，在橙色魔法罐中富集
        * 吸收橙羊羊并转化为10点橙色羊羊能量保存在罐内 
        * 消耗10点橙色羊羊能量，使一只小动物成长(不支持年幼形态右键有交互的生物，如马、羊驼)
        * 最多保存50点橙色羊羊能量
* 紫红色
    * 特殊方块：TNT
    * 合成消耗羊羊能量：100点白色羊羊能量
    * 用途：
        * 想要付出的心情，变为紫红色的烟花盛放
        * 吸收紫红羊羊并转化为15点紫红羊羊能量保存在罐内 
        * 消耗1点紫红羊羊能量，向前发射一只紫红色烟花
        * 最多保存50点紫红羊羊能量
* 海蓝色
    * 特殊方块：青金石块
    * 合成消耗羊羊能量：50点白色羊羊能量
    * 用途：
        * 海蓝的笔，如大海一般多姿
        * 吸收海蓝羊羊并转化为10点海蓝羊羊能量保存在罐内 
        * 消耗1点海蓝羊羊能量，右击羊羊或方块改变其颜色（目前不支持旗帜、染色玻璃板、床、潜影盒）
        * 对空气右击改变循环画笔颜色
        * 最多保存50点海蓝羊羊能量
* 黄色
    * 特殊方块：萤石块
    * 合成消耗羊羊能量：100点白色羊羊能量 100点浅灰羊羊能量
    * 用途：
        * 引你不断前行的，是一盏金黄的灯
        * 吸收黄羊羊并转化为10点黄色羊羊能量保存在罐内 
        * 消耗5点黄色羊羊能量，高亮周围怪物20s
        * 最多保存50点黄色羊羊能量
* 柠檬
    * 特殊方块：堆肥桶
    * 合成消耗羊羊能量：100点白色羊羊能量 50点浅灰羊羊能量
    * 用途：
        * 柠檬色的甘霖，那是生长的力量
        * 吸收柠檬羊羊并转化为10点柠檬羊羊能量保存在罐内 
        * 消耗2点柠檬羊羊能量，使面前一格作物成熟（目前不支持竹子、紫紫花、海带、浆果）
        * 最多保存50点柠檬羊羊能量
* 粉色
    * 特殊方块：西瓜块
    * 合成消耗羊羊能量：100点白色羊羊能量 100点浅灰羊羊能量
    * 用途：
        * 谁又能想到粉色拥有治愈的力量呢
        * 吸收粉羊羊并转化为10点粉色羊羊能量保存在罐内 
        * 消耗5点粉色羊羊能量，丢出一瓶治愈药水
        * 最多保存50点粉色羊羊能量
* 灰色
    * 特殊方块：铁块
    * 合成消耗羊羊能量：100点白色羊羊能量
    * 用途：
        * 灰色站在了创造的对立面
        * 吸收灰羊羊并转化为10点灰色羊羊能量保存在罐内 
        * 消耗1点灰色羊羊能量，挖掘面前3*3的石头方块
        * 最多保存50点灰色羊羊能量
* 浅灰
    * 特殊方块：地狱疣块
    * 合成消耗羊羊能量：100点白色羊羊能量
    * 用途：
        * 与白羊羊相似，却不够纯洁。正因不够纯洁，而显的更加高贵
        * 吸收浅灰羊羊并转化为5点浅灰羊羊能量保存在罐内
        * 最多保存100点浅灰羊羊能量
* 青色
    * 特殊方块：海晶石
    * 合成消耗羊羊能量：100点白色羊羊能量 50点橙色羊羊能量
    * 用途：
        * 青色的春天到了，又到了交配的季节
        * 吸收青羊羊并转化为10点青色羊羊能量保存在罐内
        * 消耗5点青色羊羊能量，使一只动物进入发情状态(暂不支持村民 不知如何实现，右键可交互动物不支持)
        * 最多保存50点青色羊羊能量
* 紫色
    * 特殊方块：紫珀块
    * 合成消耗羊羊能量：100点白色羊羊能量 100点浅灰羊羊能量 50点灰色羊羊能量 
    * 用途：
        * 红蓝之间，取而不破
        * 吸收紫羊羊并转化为2点紫色羊羊能量保存在罐内
        * 消耗50点紫色羊羊能量，获得面前方块的核心掉落物（而不破坏它）
        * 最多保存50点紫色羊羊能量
* 蓝色
    * 特殊方块：装满水的炼药锅
    * 合成消耗羊羊能量：50点白色羊羊能量
    * 用途：
        * 碧蓝如水，清泉涌起
        * 吸收蓝羊羊并转化为10点蓝色羊羊能量保存在罐内
        * 消耗2点蓝色羊羊能量，在面前放置一桶水
        * 消耗1点蓝色羊羊能量，将面前的水收回
        * 最多保存50点蓝色羊羊能量
* 棕色
    * 特殊方块：蘑菇茎干方块
    * 合成消耗羊羊能量：100点白色羊羊能量 100点浅灰羊羊能量
    * 用途：
        * 谁又能想到棕色拥有混乱的力量呢
        * 吸收棕羊羊并转化为10点棕色羊羊能量保存在罐内
        * 消耗5点棕色羊羊能量，丢出一瓶伤害药水
        * 最多保存50点棕色羊羊能量
* 绿色
    * 特殊方块：苔石砖
    * 合成消耗羊羊能量：100点白色羊羊能量 50点柠檬羊羊能量
    * 用途：
        * 这该死的绿色，竟是森林最大的敌人
        * 吸收绿羊羊并转化为10点绿色羊羊能量保存在罐内
        * 右击树根部，将一颗树砍倒，每块木头消耗1点绿色羊羊能量
        * 最多保存50点绿色羊羊能量
* 红色
    * 特殊方块：岩浆块
    * 合成消耗羊羊能量：100点白色羊羊能量 50点浅灰羊羊能量
    * 用途：
        * 将这鲜红注入矿石，得到双倍奉还
        * 吸收红羊羊并转化为10点红色羊羊能量保存在罐内
        * 消耗3点红色羊羊能量，将面前的铁矿石或金矿石烧炼为两个金属锭
        * 最多保存50点红色羊羊能量
* 黑色
    * 特殊方块：龙蛋
    * 合成消耗羊羊能量：100点白色羊羊能量 50点紫色羊羊能量
    * 用途：
        * 神秘的黑色，幽灵般的交互
        * 吸收黑羊羊并转化为5点黑色羊羊能量保存在罐内
        * 首次使用时消耗50点黑色羊羊能量，预设一个传送点，此后不可改变
        * 再次使用时消耗50点黑色羊羊能量，回到预设传送点
        * 最多保存50点黑色羊羊能量

### 羊羊魔法剪

#### 介绍
注入羊羊能量后的魔法剪刀，能从羊羊身上剪下更高级副产品，每次使用需消耗羊羊能量。

#### 获得方式
使用羊羊注魔台合成。
使用羊羊大百科魔法书右击地面上的一把新剪刀，消耗50点羊羊能量即可合成。

#### 用途
针对不同颜色的羊羊，其产品也有所变化。若身上没有足够的白色羊羊能量，则产出羊毛。

* 白色
    * 产品：骨块*1
    * 白色羊羊能量消耗：2

### 羊羊魔法剑

#### 介绍
注入羊羊能量后的魔法剑，身上能量越高伤害越高，攻击消耗羊羊能量。
