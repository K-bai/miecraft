# 羊羊工艺 Miecraft

## 简介

羊羊，世界上一种普普通通的生物。经羊羊科学家最新的研究发现，他们身体里拥有与众不同的强大能量。而它们身上多彩的颜色，正是它们体内力量的体现。羊羊科学家们不断地研究，终于发现了和平利用羊羊能量的方法。

## 物品设定

### 羊羊大百科魔法全书
#### 介绍
羊羊科学家们研究成果的结晶，只要拥有了它就能掌控羊羊们体内的能量。

#### 获得方式
合成一本书，丢给羊羊，它们吃掉后会吐出包含自身颜色的书。16种颜色集齐后便成为羊羊大百科全书。

#### 用途
用于羊羊魔法物品的能量注入。




### 羊羊魔法罐
#### 介绍
能够保存并转化羊羊能量的魔法罐。仅能够吸收并存储特定颜色的羊羊。

#### 获得方式
搭建3\*4\*3多方块结构，由最顶层至底层结构如下：


```
阶 阶 阶
阶 讲 阶
阶 阶 阶
```

```
玻 玻 玻
玻 毛 玻
玻 玻 玻
```

```
玻 玻 玻
玻 特 玻
玻 玻 玻
```

```
玻 玻 玻
玻 玻 玻
玻 玻 玻
```

其中：讲->讲台，阶->任意台阶（台阶高的一面向内，外低内高），玻->玻璃，毛->对应颜色羊毛，特->对应特殊方块（见下）。

把羊羊大百科魔法书放到顶部讲台即可合成，不同颜色合成需求及功能见下。

#### 用途
不同颜色的羊羊魔法罐用途均有不同，在使用前需要用羊羊魔法罐右击对应颜色羊羊来吸收他们。

* 白色
    * 特殊方块：骨块
    * 合成所需羊羊能量：无
    * 用途：
        * 最基础且纯净的羊羊魔法罐
        * 吸收白羊羊并转化为10点白色羊羊能量保存在罐内
        * 最多保存100点白色羊羊能量
* 橙色
    * 特殊方块：干草块
    * 合成消耗羊羊能量：100点白色羊羊能量 50点柠檬羊羊能量
    * 用途：
        * 橙色是成长的颜色，在橙色魔法罐中富集
        * 吸收橙羊羊并转化为10点橙色羊羊能量保存在罐内 
        * 消耗10点橙色羊羊能量，使一只小动物成长(不支持年幼形态右键有交互的生物，如马、羊驼)
        * 最多保存50点橙色羊羊能量
* 紫红色
    * 特殊方块：TNT
    * 合成消耗羊羊能量：100点白色羊羊能量
    * 用途：
        * 想要付出的心情，变为紫红色的烟花盛放

    * 特殊方块：青金石块

    * 用途：
        * 海蓝的笔，如大海一般多姿
        * 吸收海蓝羊羊并转化为10点海蓝羊羊能量保存在罐内 
        * 消耗1点海蓝羊羊能量，右击羊羊或方块改变其颜色（目前不支持旗帜、染色玻璃板、床、潜影盒）

## 多方块结构
### 羊羊注魔台
#### 介绍
为各种物品注入羊羊能量的装置，强化该物品的能力。

#### 搭建方式
地面用石砖搭建3*3的平台，外围一圈书架（共16个），书架上方放置16色地毯各一个，中心石砖上方放置讲台，在讲台上放上羊羊大魔法百科全书即可。

#### 用途
为各种物品注入羊羊能量。

首先将被注入物品丢到讲台上方，随后出现粒子效果，物品被锁定。站在讲台上超过1秒即可取下。

在16色羊毛的对应颜色上丢上装有对应颜色羊羊能量的魔法罐。

最后右键讲台即可开启注魔过程。


### 高级羊羊注魔台
#### 介绍
为各种物品注入羊羊能量的装置和物质精华，进一步强化该物品的能力，注魔速度较慢。

#### 搭建方式
地面用黑曜石搭建5*5的平台，外围一圈石砖（共24个），再外围一圈书架（共32个），书架上方放置16色地毯各一个（空一格），最外侧黑曜石上方放置8块板砖（空一格），中心黑曜石上方放置讲台，在讲台上放上羊羊大魔法百科全书即可。

#### 用途
为各种物品注入羊羊能量和物质精华。

首先将被注入物品丢到讲台上方，随后出现粒子效果，物品被锁定。站在讲台上超过1秒即可取下。

在半砖上方放置所需物品，在16色羊毛的对应颜色上丢上装有对应颜色羊羊能量的魔法罐。

最后右键讲台即可开启注魔过程。

### 羊羊魔法剑
#### 介绍
* 金质
    * 注入黄色羊羊能量后的魔法剑，击中敌人掉落经验，消耗黄色羊羊能量
* 钻石
    * 注入浅灰羊羊能量后的魔法剑，身上能量越高伤害越高，攻击消耗浅灰羊羊能量。

#### 获得方式
使用羊羊注魔台合成。

* 金质
    * 核心物品：金剑
    * 辅助物品：羊羊能量缓冲罐\*3 羊羊能量控制器\*1 黄能量块\*5 生物精华\*5 漏斗\*2
    * 羊羊能量：50点黄色羊羊能量 10点青色羊羊能量 10点柠檬羊羊能量
* 钻石
    * 核心物品：钻石剑
    * 辅助物品：高级羊羊能量缓冲罐\*3 高级羊羊能量控制器\*1 蓝能量块\*10 黑曜晶体\*1 木剑\*1 石剑\*1 铁剑\*1
    * 羊羊能量：100点浅灰羊羊能量 10点红色羊羊能量 10点紫红羊羊能量

#### 用途
* 金质
    * 击中一次敌人掉落3点经验
    * 每击中一次消耗1点黄色羊羊能量
* 钻石
    * 随浅灰羊羊能量变化而变化
    * 0，无攻击力，产生能量反噬
    * 0以上攻击力为浅灰羊羊能量的十分之一
    * 150能量以下每攻击一次消耗1点浅灰羊羊能量
    * 150-200消耗2点
    * 200-250消耗4点
    * 250-300消耗8点
    * 300+消耗20点
